## 🚀 Installation Rapide

### Prérequis
- **Node.js** 18+ - [Télécharger](https://nodejs.org/)
- **pnpm** (recommandé) - [Installer](https://pnpm.io/installation)

### Setup
```bash
git clone <repository-url>
cd <repository>
pnpm install
pnpm dev
```

**App accessible sur:** http://localhost:5173
